PureMVC-Plus-Plus by efnx

A MVC application architecture for C++, based off of the popular PureMVC

This is a C++ port of PureMVC. For a better description of what PureMVC is, go here -> http://puremvc.org/content/view/67/178/

Dependencies

STL

Install

Include pmvcpp.h in your main c++ source file and link with the compiled pmvcpp.cpp. There are some examples in the examples directory.
